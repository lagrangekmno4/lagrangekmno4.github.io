Welcome to Lagrangekmno4‘s Blog！这是我的第二代公开的个人博客，欢迎您的来访！

2022年初，由于初入大学的我心里颇不宁静，却因不善言谈而缺乏沟通的契机，我开始使用 hexo 博客在网络上记录一些文字，包括个人的一些想法、学习生活的一些记录，这样的记录一直持续到了现在。但事实上，在 2023 年初，我关停了网站的 github 部署，这是因为如果文字一旦被公开，自己写下的文字便会考虑到读者的因素，很多文字记录的便不是自己最真实的想法；加之当时疫情刚刚结束，我希望我不再依赖网络和文字来记录自己的生活，但事实上并没有改变太多（笑）。前些天在记录时又在想另外一件事情：

所以说自己每次阅读自己写过的东西都会感觉有那么一丝焦虑和压抑，其实无非是留给自己这样一个释放和表达压抑和焦虑的空间，而更多地去做一些自己更能掌控的、乐观的和积极的事情。从这个角度来看，定期写文字的工作也是非常有价值的。只不过可能把这样的交流都通过自己在电脑上敲文字可能会产生一些迭代式的自我批判和剖析，容易产生一些钻牛角尖之类的情况，类似“我想不明白这个，我要想尽一切办法去想明白“很多情况下不如”想不明白就不想了，等一个预期之外的机会来解决这一切“。

在留给自己一个封闭虚拟思想平台的同时同时也要留一个对外的平台，尽管他仍然是虚拟的，但仍然拥有意义，这也是我恢复这个网站的原因之一。当然，也有一些其他值得我重启的理由，比如我能写的东西的类型更多了，无论是数学还是音乐，还是其他的东西，都总要比去年那个只会感叹时运不齐的忧郁记录要好了太多，等等。

第一代公开的个人博客做到了每天记录，但事实上无非是倒一些焦虑和批判时事的垃圾，现在想来，那些文字无非是排解内心郁闷的借口，不足为外人道也。新版的个人网站则会记录一些“足为外人道也”的内容，如刚才所说，包括数学和音乐的学习记录，以及一些经历与见闻，总之，我希望我能写下一些有趣和有价值的文字，作为虚拟世界里属于自己的对外平台和广播。

当然，我仍然更希望拥有更多的对外的平台。网站开设了评论区，并且为了降低壁垒，我使用了可以不登录 github 网站的 valine 评论平台，您可以选择性的填写信息。如果您愿意就我网站中的部分内容进行讨论，可以通过预留的邮件来联系我，我大概率会在两小时之内通过邮件或者添加您的联系方式与您取得联系。同时，事实上即使是我“对内”的思想平台也并非完全在本地保存，如果您对此感兴趣，也可以向我索要相关的网址。

最后，我可能在现实生活中并不是一个健谈的人，但这个网站里的我大概率会很健谈，如果您觉得我还算一个比较有意思的人，也希望您能够常来。
